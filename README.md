# 🛒 MyBasket

MyBasket is a simple e-commerce web application that allows users to browse products and manage their shopping experience.

## 🚀 Features
- Product listing
- Add to cart
- Responsive UI
- Basic order flow

## 🧱 Tech Stack
- Frontend: HTML, CSS, Tailwind CSS, JavaScript
- Backend: Spring Boot (planned)
- Database: MySQL (planned)
